## 🚀 Selected Work

### [Nova PPI Capixaba (RemanejaSUS)](https://sites.google.com/view/novappicapixaba/)

**Problem**

Monthly statewide healthcare resource allocation required extensive manual coordination among municipalities.

**Solution**

Designed and implemented the complete RemanejaSUS platform, automating data collection, validation, negotiation workflows, reporting, and monthly allocation cycles.

**Stack**

AppSheet • Google Apps Script • Google Sheets • Looker Studio

**Results**

- Automated statewide monthly allocation cycles.
- Increased transparency and auditability.
- Prevented financial inconsistencies before approval.
- Standardized negotiation workflows across municipalities.
